1. Системные требования

Минимум: 
1. Процессор класса 486 DX-4 с тактовой частотой 100 мегагерц
2. 16 мегабайт оперативной памяти
3. Около 3 мегабайт свободного дискового пространства
4. Windows 98
5. SVGA монитор 800х600, 256 цветов минимум.

Рекомендуется:
1. Процессор класса Pentium с тактовой частотой 200 мегагерц
2. 32 мегабайт оперативной памяти
3. Около 5 мегабайт свободного дискового пространства
4. Windows 98 или выше
5. SVGA монитор 16 миллионов цветов 800х600.



2. Обновление версий

Внимание! Обновлению подлежат версии 4.0 Alpha 1 и выше! Для обновления игры установите игру в каталог старой версии.
