Это базовое руководство по хакерству с использованием модема. Оно не претендует на полноту изложения материала. Для более глубокого изучения отсылаю вас к консольным командам, справку по которым можно получить введя команду help имя_команды.

Здесь и далее предполагается наличие у вас следующих данных:

  1. Номер провайдера для довзона. Обозначение - number;
  2. Логин для доступа в интернет - login;
  3. Пароль - pass.

Порядок операций:

  1. Сбрасываем настройки модема командой ATZ;
  2. atdt number;
  3. Появится запрос на ввод логина. Вводим login;
  4. pass;
  5. Выполняем команду ipconfig для того чтобы узнать свой адрес;
  6. ping-аем любой IP;
  7. Если есть ответ выполняем команду ports;
  8. Затем rcontrol;
  9. Запускаем файловый менеджер.
